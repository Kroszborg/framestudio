# FrameStudio Design System

## Brand
- **Name**: FrameStudio
- **Tagline**: Professional video editing, anywhere.
- **Personality**: Precision tool, dark & cinematic, professional but approachable

## Colors
- Background: `#09090F`
- Surface: `#12121E`
- Surface Elevated: `#1A1A2E`
- Surface Active: `#242440`
- Border: `#2A2A3E`
- Border Light: `#353550`
- Primary: `#6366F1` (Indigo)
- Primary Light: `#818CF8`
- Primary Muted: `rgba(99,102,241,0.15)`
- Accent: `#A78BFA` (Violet)
- Text: `#F1F1F6`
- Text Secondary: `#8B8BA3`
- Text Muted: `#5C5C72`
- Success: `#22C55E`
- Danger: `#EF4444`
- Warning: `#F59E0B`
- Timeline BG: `#0D0D18`

## Typography
- **Display**: System default (San Francisco / Roboto)
- **Body**: System default
- **Mono**: Monospace (for timecodes)
- Sizes: 10, 12, 13, 14, 16, 18, 20, 24, 28, 32

## Spacing
- xs: 4, sm: 8, md: 12, lg: 16, xl: 20, 2xl: 24, 3xl: 32

## Radius
- sm: 6, md: 8, lg: 12, xl: 16, full: 999

## Shadows
- Minimal — rely on surface color elevation instead
- Glow effects for active/selected states using primary color

## Components
- Dark surfaces with subtle borders
- Active states use Primary Muted background + Primary border
- Icons: Phosphor, regular weight (20px default), bold for active tabs
- Buttons: filled primary, ghost secondary, icon-only for toolbar
- Timeline clips: colored blocks with thumbnail strips
- Bottom sheets: Surface Elevated bg, handle bar indicator
